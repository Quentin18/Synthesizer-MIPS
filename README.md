# Synthétiseur MIPS

Instructions :
- Pour jouer une note : appuyer sur une touche du clavier
- Pour changer d'instrument : appuyer sur &, puis entrer le numéro de l'instrument, et presser ENTER
- Pour quitter le programme : appuyer sur SPACE
